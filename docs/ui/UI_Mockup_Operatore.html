nu miezz a piacer — Mockup Interfaccia Operatore

Sketch a bassa fedeltà delle schermate del profilo operator. Ogni schermata mostra il frame mobile (340px) e desktop (760px) affiancati, seguito dalla documentazione: note, funzioni → slice, slice VSA e componenti shadcn. Navigazione via BottomNav (5 tab: Mappa · Manutenzione · Ticket · Scadute · Altro). Login condiviso col cliente (fuori scope). Mappe = placeholder, implementazione con Leaflet + OSM. Note

  • BottomNav è componente custom (non shadcn) — già esiste in apps/web/src/layout/BottomNav.tsx, va esteso con varianti per ruolo (customer/operator/admin).
  • 5 tab max per leggibilità. Voci secondarie (GPS, Bonus, Utenti, Profilo, Logout) nel tab “Altro” → apre Sheet shadcn dal basso.
  • Dettaglio Veicolo e Pannello Segnalazioni NON sono tab: sono sheet contestuali sopra la mappa.
  • Voci BottomNav filtrate da getVisibleSlices('operator') sul sliceRegistry.
