apps/web/src/slices/EndRide/
Entry point: apps/web/src/app/rides/end/page.tsx

Differenza con /rides/active

/rides/end è raggiunto direttamente dal banner "Termina" nella home quando la corsa è ACTIVE.
/rides/active (PauseRide) è il pannello completo con pausa/ripresa/termina.

API
  • POST /api/rides/:id/end — hook useEndRide()
  • Body: { userId, endLocation: { lat, lng }, distanceKm }
  • Coordinate e distanza hardcoded demo
  • Risposta: { ride, totalCost }
Stato globale
  • Legge activeRide da useAppStore per mostrare i dati della corsa
  • Dopo il successo: clearSession() per resettare tutto lo store
  • Se activeRide è null → mostra "Nessuna corsa attiva" (stato guard)
Calcolo costo (lato API)

Il costo viene calcolato dal backend:
unlockFee + (durationMinutes × perMinuteRate) + (distanceKm × perKmRate)
I dati di pausa vengono inclusi nel calcolo se presenti.
