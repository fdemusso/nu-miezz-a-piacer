Route: /search
Slice

apps/web/src/slices/Search/
Entry point: apps/web/src/app/search/page.tsx

API
  • GET /api/vehicles/search?q=&type=&onlyAvailable=
  • Hook useSearch(filters, submitted) — la query parte solo dopo che submitted = true
  • Risposta: { total, vehicles: SearchVehicleItem[] }
  • SearchVehicleItem è definito in @mvp/contracts
Stato locale (no store)
  • filters: SearchFiltersState — { query, type, onlyAvailable }
  • submitted: boolean — controlla se la ricerca è attiva
  • Il bottone "Reset" azzera entrambi i campi
Filtri
  • query — testo libero su modello o targa (con bottone ✕ di clear)
  • type — chip row scrollabile: Tutti | Scooter | Bici | Moped | Auto
  • onlyAvailable — toggle chip "Solo disponibili"
  • Enter nel campo di testo triggera la ricerca
Stati UI
  • initial icona Search + "Cerca un veicolo"
  • loading 3 skeleton card
  • results lista VehicleCard linkate a /vehicles/:id
  • no results icona + "Nessun risultato"
  • error AlertCircle + "Errore"
Navigazione
VehicleCard click → /vehicles/:id
